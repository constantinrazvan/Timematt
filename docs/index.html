get started

Running on your machine in a few minutes

Clone the repo, point appsettings.json at your own MySQL database, apply the migrations, and run it.

bash 
# clone
git clone https://github.com/constantinrazvan/Timematt.git
cd Timematt/TimeMatt

# configure your database in appsettings.json, then
dotnet ef database update
dotnet run
