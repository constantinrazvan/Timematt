Features

Everything a freelancer's workflow needs

One workspace instead of five different SaaS tabs.

C

Clients

A CRM-style list of clients with per-client revenue, hours and project history.

P

Projects

Track status, fixed-price or hourly billing, files and comments per project.

T

Tasks

A drag-and-drop Kanban board, plus list and grid views, with a detail page per task.

⏱

Time Tracking

Start, pause and stop a timer against a specific project and task, with daily/weekly/monthly summaries.

R

Reports

Hours and revenue charts, with one-click PDF and CSV export of the whole workspace.

A

Real Auth

ASP.NET Core Identity with roles, forgot/reset password, and a client invite flow delivered by email.

Tech Stack

Built on a boring, reliable stack

No exotic dependencies — just ASP.NET Core, Razor and a real database.

ASP.NET Core MVC (.NET 10) Razor Views Bootstrap 5 Entity Framework Core 9 MySQL / MariaDB ASP.NET Core Identity Chart.js SortableJS

Self-host it in a few minutes

Clone the repo, point it at your own MySQL database, and run it.

# clone
git clone https://github.com/constantinrazvan/Timematt.git
cd Timematt/TimeMatt

# configure your database in appsettings.json, then
dotnet ef database update
dotnet run
