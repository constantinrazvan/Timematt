Get started

Running on your machine in minutes

Clone the repo, point appsettings.json at your own MySQL database, apply the migrations, and run.

Get the code Read the docs
bash — TimeMatt 
# clone
git clone https://github.com/constantinrazvan/Timematt.git
cd Timematt/TimeMatt

# configure your database in appsettings.json, then
dotnet ef database update
dotnet run
